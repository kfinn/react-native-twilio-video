# react-native-twilio-video

react-native bindings for Twilio Video

## Installation

```sh
npm install react-native-twilio-video
```

## Usage

```js
import TwilioVideo from "react-native-twilio-video";

// ...

const result = await TwilioVideo.multiply(3, 7);
```
